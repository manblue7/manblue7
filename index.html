HTML

HTML

Hyper text markup language. Markup refers to a way of annotating text that is distinguishable from the text itself


Start file with:

  • <!DOCTYPE html>
  • <html lang="en"> (all content goes in here)

Head elements:

  • <head>
  • <meta>
  • <link>
  • <title>

Some body elements:

Some elements are default block level or inline. Block spans the width of the browser by default. Inline elements dont start a new line/section, they are contained within the block elements. Some are marked whether they are block or inline

  • <main> (block)
  • <nav> (block) semantic
  • <section> (block) semantic
  • <header> (block)
  • <article> (block) semantic
  • <div> (just a generic block/container stands for document division)
  • <aside> (tangentially related to parent container)
  • <span> (kind of like a div? but for inline)
  • <h1> (header, h1 - h6) 1 being largest
  • <p> (block)
  • <a> (link, inline)
  • <strong> (bolds text, inline)
  • <em> (italicizes, inline)
  • <ul>
  • <ol>
  • <li>
  • <table>
  • <form>
  • <img> (inline)
  • <dialog>
  • <footer> (block)
  • <button>
  • <script>

CSS

CSS

Cascading Style Sheets. Used to style the appearance of a website.

CSS declarations
property: value

  • font-family: ariel, sans-serif(fallback);
  • font-size: 20px
  • font-weight: bold;
  • text-align: center;
  • color: blue/rgb(a)(red, green, blue(0-250), hsl(a)(hue(0-360), satur., lightness, ((a)opacity0-1)); ((a)opacity 0-1)/Color can be hex like #FFFFFF
  • letter-spacing: 2px;
  • text-transform: uppercase;
  • word-spacing: 0.2em (any value != normal is added to word spacing)
  • letter-spacing: 0.02em;
  • line-height: 1.2(unitless num based on current font size)/px/%/em/rems (increases vertical spacing between lines of text)

  • list-style: square;
  • background-color: pink;
  • opacity: 0.5;
  • background-image: url("");
  • background-attachment: fixed;
  • background-size: cover;

  • :hover (when mouse is hovering over element)

  • height: 150px;
  • width: 58%;
  • display: inline-block/none/block/inline/flex/inline-flex; (dictates if element can share horizontal space)
  • box-sizing: border-box;
  • padding: 10px;
  • margin: 20px auto;
  • border: 3px solid cyan;

    • And many more, check notes file or online resource
